# WiFinder-nu11secur1tyAI

A Python-based utility for Windows 11 to retrieve saved Wi-Fi networks and passwords.

## Features
*   **Auto-Privilege Elevation**: Requests Admin rights automatically.
*   **Multi-Language**: Works on English and Bulgarian Windows systems.
*   **Clean Output**: Formatted table for SSIDs and Passwords.

## Usage
```powershell
python wifinder.py
```
